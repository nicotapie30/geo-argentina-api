# GeoArgentina API — Frontend

Sitio de documentación para la [GeoArgentina API](https://github.com/nicotapie30/geo-argentina-api), una API REST pública con datos geográficos completos de Argentina.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Desarrollo local

```sh
git clone https://github.com/nicotapie30/geo-argentina-api-front
cd geo-argentina-api-front
npm install
npm run dev
```

## API

Base URL: `https://geoargentinaapi.up.railway.app/v1`

Recursos disponibles: provincias, departamentos, localidades.
Sin autenticación · JSON · 120 req/min.
